Work

What I've done.

Here's some selections of work that I've done. All descriptions are in brief summary. Do not hesitate to contact me for more info of what I've done throughout each projects.

Roda Fantasi

Strategy Word Game

Game that I've created in a team of 3 programmers during my internship. It is up on the android and iOS app store. This game requires the player to guess the correct word that is generated from the game database. Player can either spin the wheel and earn game currencies if they guessed the correct letter in the word or buy vowels to get more insight on is the word generated.

Kluno: A Kid's Dream

Action Puzzle Game

A fast paced match three game. This first intellectual property by Gameka combines both tower defense and match three mechanic to create a brand new fast paced game. Focuses on the backend side, responsible for reading and writing data to the server, getting leaderboard and also login using google play or game services.

Kluno: Oogly's Escape

Action Puzzle Game

Inspired by Kluno: A Kid's Dream come Kluno Oogly's Escape with a whole new different gameplay and mechanic. My role is to help out with the backend and also the front end of the game. Learned a lot about Unity UI system here which includes Unity Grid layout, anchoring, and also scroll rect during the production of this game. Uses a lot of Unity animation system and also animator.

Kluno: Hero Battle

Action Puzzle Game

Not a kid nor an escape, is a HERO BATTLE! Kluno: Hero Battle is a realtime multiplayer game that is fast paced thus required a very high concentration and agility in order to excel. Focuses on the backend and the front end part of the game. During production period, practices and learned a lot of Object-oriented programming and how to make dynamic script in this project. Also learned on how to set-up and publish a game to google play store and apple store. Did some of the mobile optimization such as script caching.

Knight Clash

2D Side-scroller Arcade

Work as a solo programmer from scratch together with artist and game designer to create this side-scroller game in 3 months time. Game optimization were done later on which fixes game crashes on low ends phone due to high usage of memory.

Space Rider: Star Hunt

Arcade runner

Work together with 2 other programmer to work on this game and complete it within 4 months time. Start to involve in some team leading and also project management. Focuses mainly on the game enemy and bullet mechanic, game level generator ,integration of telco related stuff such as subscription which involves API calling.

Super Jones

Endless Arcade

An endless arcade game where the player will need to drill as deep down as possible before the timers run out and achieve as much score as possible by doing combos and drilling gems on the way down. Focuses mainly on the game UI, game polishing and effects implementation into the game. Completion of the game in 4 months time. Guide team onto achieving deadline and also project planning to achieve a polished game state. This game was nominated in LEVEL UP KL 2020 as one of the finalist.

Kingdom Battles PvP

Strategy Tower Defense

A multiplayer game where player will be able to matchmake and find other players base while building a defensive base in order to prevent other players from successfully invading our base which will cause trophy deduction. Work mainly on the game UI, cards/gacha logic, tutorial, player inventory management, and also API integration. Completion of game in 4 months time.

Ghost Force

Arcade

Pattern detection arcade game. Destroys and kill all the ghost by drawing the pattern on the ghost and also collect combos to be able to use skills. Focus mainly on the backend of the game which includes API creation, saving data and also return data to the client with some basic encryption and decryption for security purposes.

P Blast

Match-2-game

A Match-2-game. Work together with a programmer and completed the game in 6 months time. Focuses mainly on the game board mechanic, tiles mechanic, power ups mechanic, animation and also editor scripting to ease game designer on level creation.

Ghost Force Multiplayer

Arcade

A realtime PVP game using photon. The objective of the game is to draw the correct lanes pattern to destroy the ghost in a lane and summons ghost to the opponent. Worked mainly on the backend side and also the pattern recognition accuracy improvised version based on an open source library.
